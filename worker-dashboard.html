🐾 Log a Colony Event
Only log when something actually happens — a birth, death, replacement, isolation, or male rotation. You don't need to log every day.
If multiple animals died, submit one entry per animal.
Use this after a death when a new animal is placed into the vacant slot. The old ID is retired; the new animal gets the next available ID.
Mark an animal as isolated (sick, aggressive, post-birth recovery). Owner sees this flagged immediately.
